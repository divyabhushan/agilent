Issue Log

The logs of all the issues raised by the users are listed and managed in this section.

Admin can filter the logs based on User Name, Issue Date, and Type of Issue.

To download the log detials in a .csv forat, select Export to CSV at the top-right corner.

Edit Issue Log#

A Power user who has an Edit Issue Log permission granted, can edit the Status and enter ABRT Comments for the issue logs. (Use the scroll bar at the bottom of the screen).
