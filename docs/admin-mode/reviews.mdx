Power users generate a review request in 2 ways: 
a. <Link to={useBaseUrl('docs/admin-mode/upload-report')}>Uploading a Report</Link>, or 
b. <Link to={useBaseUrl('docs/admin-mode/upload-report#report-review')}>Triggering a Report Review</Link> manually. 

The Reviewer (another Power user) can view the list of pending reports by selecting the **Reviews** section in the left navigation pane. 

The Reviewer can approve ( <img height="20" src={useBaseUrl('img/check-circle.svg')} /> ) or reject ( <img height="20" src={useBaseUrl('img/x-circle.svg')} /> ) the report in the **Action** menu.
Reviewer must enter relevant comments to approve/reject the report.
