Admin/Power user can get the usage metrics of how the portal is used based on different parameters.
User can select the combinations of metrics to fetch the respective usage report and click **Submit Query**.

## Web Metrics

Shows a log of the the Web reports usage:

<div class="center">
  <Zoom>
    <img
      alt="Web Metrics"
      src={useBaseUrl("admin/web-metrics.png")}
    />
  </Zoom>
  <p>Web Metrics</p>
</div>

Following are the parameters:

1. **Select Metrics**: Select a sub category from the access parameters label.
1. **Select Reports**: A list of all the Web reports.
1. **From date**: Select a beginning period.
1. **To date**: Select the latest period. 
1. **Select User Type**: ABRT or non ABRT user.
1. **Select Segments**: Select the Segments for usage activity.

Click [here](./assets/Metrics.xlsx) to download the complete Metrics sheet.
