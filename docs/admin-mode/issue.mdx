A log of all the <Link to={useBaseUrl('docs/user-mode/contact-us')}>issues raised by the users</Link> are listed and managed in this section.

screenshot

Admin can filter the logs based on **User Name, Type**, and **Status**, and change the status to "Open", "Pending", or "Closed".
Click **Export to CSV** to download the log details in a `.csv` format.
